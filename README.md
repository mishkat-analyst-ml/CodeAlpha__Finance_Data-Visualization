# CodeAlpha__Finance_Data-Visualization
Finance data analysis project using(pandas,Matplotlib,seaborn) to explore investment behavior,preferences, and financial decision-making patterns.

## Project Overview
This project focuses on analyzing a finance dataset to understand investment behavior, preferences,and decision-making patterns among individuals.
This goal is to transform raw data into meaningful insights using Exploratory Data Analysis (EDA) and **data visualization techniques**.

## Objectives
Analyze investment patterns based on age and gender.
identify the most preferred investment avenues.
understand risk behavior and return expectations.
Explore reasons behind choosing equity, bonds, and fixed deposits.
Determine how people monitor investments.

## Tools and Technologies

.Pandas 
.Matplotlib,seaborn

## Key insights
. Majority of the individuals prefer investing over saving only.
.Mutual Funds and Equity Market are the most popular investment options.

.Expected return are generally modearte **(10%-30%)**
.The internet are the primary source of fianacial information.

This project demonstrates how data visualization can help uncover investment trends,user behavior, and financial descions patterns, which can support better business and financial strategies.
